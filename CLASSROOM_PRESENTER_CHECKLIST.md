# Classroom Presenter Checklist

## /classroom/s01/presenter

```txt
□ Presenter View header 표시
□ Student Deck 버튼 표시
□ Slide Overview 버튼 표시
□ Teacher Board 버튼 표시
□ 전체 슬라이드 목록 표시
□ 현재 슬라이드 정보 표시
□ Teacher Note 표시
□ Next Slide 미리보기 표시
□ Class Timer 표시
□ Timer Start 작동
□ Timer Pause 작동
□ Timer Reset 작동
□ Back / Next 버튼 작동
□ ArrowLeft / ArrowRight / Space 키 작동
```

## /classroom/s01/overview

```txt
□ 전체 슬라이드 카드 표시
□ 각 카드에 title / subtitle / teacher note 표시
□ visualType 표시
□ focusText 표시
□ Open Deck 버튼 작동
□ Presenter View 버튼 작동
```

## Regression

```txt
□ /classroom/s01 작동
□ /teacher/s01에서 Presenter View 버튼 표시
□ /admin/qa에 presenter/overview route 포함
□ /game/s01 작동
□ /sessions 작동
□ npm run build 성공
```
