# Classroom Mode Checklist

## /classroom/s01

```txt
□ Classroom Mode header 표시
□ Slide 1 Hello Song 표시
□ Next 버튼 작동
□ Back 버튼 작동
□ Space / ArrowRight 이동 작동
□ ArrowLeft 이동 작동
□ Fullscreen 버튼 작동
□ Teacher Note 표시
□ Slide action button 표시
□ video slide에서 mp4 placeholder/재생 영역 표시
□ audio slide에서 audio control 표시
□ game slide에서 Start Game 버튼 표시
□ mission slide에서 Worksheet 버튼 표시
```

## /teacher/s01

```txt
□ Open Classroom Mode 버튼 표시
□ 버튼 클릭 시 /classroom/s01 이동
```
