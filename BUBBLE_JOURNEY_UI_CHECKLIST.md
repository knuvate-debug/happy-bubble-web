# Bubble Journey UI Checklist

## /sessions

```txt
□ Bubble Journey 제목이 보이는가?
□ Start Today’s Bubble 버튼이 보이는가?
□ S1이 open bubble로 가장 눈에 띄는가?
□ S2~S15가 coming soon bubble로 부드럽게 보이는가?
□ Bubble Status Legend가 보이는가?
□ 모바일에서 세션 버블이 한 줄로 읽히는가?
□ 데스크탑에서 지도처럼 보이는가?
```

## /sessions/s01

```txt
□ Today’s Bubble Hero가 보이는가?
□ S, A, T가 크게 보이는가?
□ Start Bubble 버튼이 보이는가?
□ Watch / Play / Sing / Mission 활동 버블이 보이는가?
□ Teacher Bubble Board 링크가 보이는가?
□ Home Bubble 링크가 보이는가?
```
