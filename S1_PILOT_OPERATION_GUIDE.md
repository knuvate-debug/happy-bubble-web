# S1 Pilot Operation Guide

## Pilot goal

```txt
S1 하나만으로 실제 6–7세 첫 영어 학습자 수업을 운영할 수 있는지 확인한다.
```

## Pilot roles

```txt
Teacher
→ Presenter View로 수업 진행

Student
→ Classroom Deck / Game Pack 이용

Parent
→ Parent Mode에서 결과 확인

Admin
→ QA / Asset / Report 상태 확인
```

## Pilot observation points

```txt
□ 아이가 글을 읽지 못해도 다음 행동을 유추하는가?
□ Game Pack 3종 중 어떤 게임에서 가장 몰입하는가?
□ Native/Korean/Bilingual/Substitute 모드의 teacher note가 실제로 충분한가?
□ 수업 시간이 30–45분 안에 들어오는가?
□ Worksheet가 수업 중 필요한가, 가정 연계가 나은가?
□ Parent 화면 설명이 보호자에게 이해되는가?
```

## After pilot

```txt
1. /teacher/report/s01 확인
2. /teacher/game-report/s01 확인
3. /admin/qa 확인
4. 아이 반응 메모
5. 강사 피드백 메모
6. S1 개선 후 S2 제작 시작
```
