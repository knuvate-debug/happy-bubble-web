# Rosebud AI + Phaser Workflow

## Recommendation

```txt
Rosebud AI = idea lab
Phaser = production engine
```

## Use Rosebud For

```txt
- 게임 컨셉 빠른 시각화
- 아이들이 좋아할 조작 흐름 테스트
- 버블/캐릭터/배경 분위기 탐색
- 미니게임 아이디어 비교
```

## Use Phaser For

```txt
- 실제 HBE 웹앱 통합
- Supabase learning_events 기록
- Teacher / Parent / Admin Report 연결
- QR 교재 연결
- S1~S15 공통 엔진화
```

## Good Rosebud Prompt Example

```txt
Create a simple bubble popping phonics game for 6-year-old Korean children learning English sounds s, a, t. The child hears a sound and taps the matching bubble. Use soft pastel bubble visuals, no text-heavy UI, large buttons, gentle feedback, and no competitive scoring.
```

## How to Bring Rosebud Ideas Back

```txt
1. Rosebud에서 컨셉 실험
2. 마음에 드는 조작과 피드백만 정리
3. HBE Game Pack 구조에 맞게 Phaser로 구현
4. Supabase 기록/Teacher Report/Parent Mode와 연결
```
