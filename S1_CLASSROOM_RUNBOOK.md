# S1 Classroom Runbook

## Before Class

```txt
1. /teacher/s1-ready에서 준비 상태 확인
2. 필수 asset missing 여부 확인
3. /teacher/s01에서 Instructor Mode 선택
4. 강사용 기기: /classroom/s01/presenter?mode=korean
5. 전자칠판/공유화면: /classroom/s01
```

## Recommended 40-minute flow

```txt
0–3 min    Hello Song
3–8 min    Today’s Bubble + Bubble Theater
8–15 min   Sound Focus: s / a / t
15–27 min  S1 Game Pack
27–32 min  Singing Bubble
32–38 min  Worksheet Mission
38–40 min  Goodbye Song
```

## If time is short

```txt
필수:
Hello Song
Today’s Bubble
Sound Focus
Sound Match
Goodbye Song

선택:
Bubble Pop
Build SAT
Worksheet
```

## If teacher is native

```txt
/classroom/s01/presenter?mode=native
```

Use short English scripts. Avoid long explanations.

## If teacher is Korean

```txt
/classroom/s01/presenter?mode=korean
```

Use Korean guidance to reduce anxiety, but keep the English sound exposure.

## If substitute teacher

```txt
/classroom/s01/presenter?mode=substitute
```

Follow the numbered teacher notes. Do not improvise long explanations.
